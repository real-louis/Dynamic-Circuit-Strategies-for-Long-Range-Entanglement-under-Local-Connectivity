# 附件：貝爾態製備、局域變換與貝爾測量（書面可直接引用）

以下與本專題 `challenge_common.generate_ladder_bell_circuit` 之 `target_state` 選項一致。  
**貝爾測量**：先 `CX(q_ctrl, q_tgt)`，再 `H(q_ctrl)`，再對兩線做 **Z 基量測**；兩個古典 bit **皆需保留**方可決定傳送／交換後之 Pauli 修正。

## 製備（相鄰兩量子位 q₀、q₁）

| 目標態 | 門序列（由左至右） | 計算基底下的關聯 |
|--------|-------------------|------------------|
| \(\|\Phi^+\rangle\) | `H(q₀)` → `CX(q₀,q₁)` | 同號：00 或 11 |
| \(\|\Phi^-\rangle\) | `H(q₀)` → `CX(q₀,q₁)` → `Z(q₀)` | 同號：00 或 11（整體相位差 \(\pi\)） |
| \(\|\Psi^+\rangle\) | `H(q₀)` → `CX(q₀,q₁)` → `X(q₁)` | 異號：01 或 10 |
| \(\|\Psi^-\rangle\) | `H(q₀)` → `CX(q₀,q₁)` → `Z(q₀)` → `X(q₁)` | 異號：01 或 10（單態） |

## 由 \(\|\Phi^+\rangle\) 經**單邊** Pauli 的局域變換（與本程式在 e0/e1 上之補償同一類）

| 目標 | 額外閘（作用在單一線上即可，實作可選 e0 或 e1） |
|------|-----------------------------------------------|
| \(\|\Phi^-\rangle\) | `Z` 在 e0 **或** 等價組合 |
| \(\|\Psi^+\rangle\) | `X` 在 e1（本程式寫法）等 |
| \(\|\Psi^-\rangle\) | `Z` 與 `X` 分別作用於兩端（本程式：`Z(e0)` + `X(e1)`） |

## 貝爾測量（診斷／糾纏交換核心）

| 步驟 | 操作 |
|------|------|
| 1 | `CX` 於待測兩量子位（控制／目標與實作約定一致） |
| 2 | `H` 於原控制線 |
| 3 | 兩線 **Z 量測** → 兩個古典 bit |

量測結果 \((m_0,m_1)\) 與後續 **X／Z** 前饋的對應請與採用之閘序對照表一致（本專題程式註解已標明 |Φ⁺⟩ 目標下之規則）。

## 3. Error budget 對照（Appendix B.2）

論文統一公式：

\[
\lambda_{\mathrm{tot}} = t_{\mathrm{idle}}\lambda_{\mathrm{idle}} + N_{\mathrm{CNOT}}\lambda_{\mathrm{CNOT}} + N_{\mathrm{meas}}\lambda_{\mathrm{meas}}, \quad \mathcal{F}_{\mathrm{proc}} \geq e^{-\lambda_{\mathrm{tot}}}
\]

| 策略 | \(t_{\mathrm{idle}}\) | \(N_{\mathrm{CNOT}}\) | \(N_{\mathrm{meas}}\) | 主導誤差來源（大 \(L\)） |
|------|----------------------|----------------------|----------------------|-------------------------|
| 靜態 GHZ | \(\sim L^2/4\) | \(2L-1\) | 0 | idle（二次） |
| 糾纏交換 | \(\sim 1 + \mu L/2\) | \(L\) | \(L-1\) | measurement + 前饋 idle |
| LRCX dynamic | \(2+\mu\)（常數） | \(L+1\) | \(L\) | measurement（crossover 條件：\(\lambda_{\mathrm{meas}} < 3\lambda_{\mathrm{CNOT}}\)） |
| LRCX unitary Ic | 0 | \(4L+1\) | 0 | CNOT（線性累積） |

參數（論文 Fig. 7）：\(\mu \approx 3.65\)，\(\lambda_{\mathrm{idle}}=0.03\)，\(\lambda_{\mathrm{CNOT}}=0.02\)，\(\lambda_{\mathrm{meas}}=0.03\)。

## 4. Crossover 對照

| 現象 | 本專題（Aer 模擬） | 論文（實機） |
|------|-------------------|-------------|
| 短距優勢 | \(L=1\)：靜態 \(P \approx 0.94\) > 交換 \(\approx 0.90\) | \(n \lesssim 10\)：unitary LRCX fidelity 較高 |
| 長距趨勢 | 靜態衰減較緩；交換快速下降 | \(n \gtrsim 10\)：dynamic LRCX 優於 unitary |
| 理論解釋 | measurement + classical latency 主導交換劣化 | 同上 + 實機 feed-forward 時間隨 bitstring 增長 |
| GHZ 製備 | 策略甲 ≈ 論文 unitary GHZ | dynamic GHZ 未見 clear crossover（Section III） |

## 6. 口試「差異化」答法

**Q：你的專題和這篇 PRX Quantum 有什麼不同？**

A：論文主軸是 **long-range CNOT gate teleportation**（up to 101 qubits）與 **GHZ state preparation**；我的競賽題是 **Bell state preparation on a ladder**。但兩者共用 **Bell measurement + Pauli feed-forward** 核心機制。我的貢獻在於：（1）在更具體的梯子競賽設定下做 fair noise comparison；（2）reproduce 論文一般電路；（3）用 error budget 統一解釋兩者的 crossover 現象。

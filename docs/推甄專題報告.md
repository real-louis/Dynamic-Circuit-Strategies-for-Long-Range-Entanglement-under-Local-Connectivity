## 摘要

在僅具局域（nearest-neighbor）連接的超導量子處理器上，製備相距甚遠的兩量子位元之間的貝爾糾纏是一項核心挑戰。本專題以 Ephys Challenge 2026「梯子佈局長程貝爾態」為問題驅動，實作並比較兩類策略：（甲）靜態 GHZ 鏈加反向 CNOT 解糾纏；（乙）糾纏交換（Bell 測量 + Pauli 前饋）。在統一 depolarizing／T1T2／讀出雜訊模型下，我們給出 CNOT 資源封閉式、CHSH 理想驗證，以及含噪 crossover 曲線。

為將競賽成果與前沿文獻對接，我們進一步 reproduce PRX Quantum 5.030339 的長程 CNOT gate teleportation（Fig. 1a）與 GHZ 動態製備（Fig. 2b），並以 pytest 驗證 statevector 等價性。最後，採用論文 Appendix B.2 的 Pauli–Lindblad error budget，統一解釋短距靜態策略優勢與長距動態策略的資源優勢及 measurement／feed-forward 瓶頸，並提出可延伸的研究展望。

**關鍵詞**：動態量子電路、長程糾纏、糾纏交換、NISQ、error budget

## 1. 引言與動機

### 1.1 NISQ 時代的連接性限制

現今超導量子處理器（如 IBM heavy-hexagonal 佈局）的量子位元僅能與鄰近節點作用。若要在距離為 \(L\) 的兩端點 \(e_0\)、\(e_1\) 之間建立糾纏，傳統 **單元（unitary）** 電路需沿鏈逐段傳遞，兩比特閘深度至少 \(\mathcal{O}(L)\)，且 idle 退相干隨深度二次累積（Lieb–Robinson 資訊光錐）。

**動態電路（dynamic circuits）** 利用 mid-circuit measurement 與 classical feed-forward，使量子資訊可經由古典通道「瞬間」傳遞，在理論上達成常數量子深度（Bäumer et al., PRX Quantum 5, 030339）。

### 1.2 本專題的研究問題

> 在梯子拓樸的局域連接限制下，如何以可擴展方式製備端點 \(e_0\)、\(e_1\) 的目標貝爾態 \(|\Phi^+\rangle\)？靜態與動態策略在 **資源** 與 **含噪保真度** 上如何權衡？

本報告以競賽解題為第一幕、論文 reproduce 為第二幕、error budget 統一分析為第三幕，構成完整研究敘事。

## 2. 問題形式化

### 2.1 拓樸與符號

- **梯子圖**：雙腳節點，雙位元閘僅能作用於圖上相鄰節點。
- **單腳抽象**：端點 \(e_0 \leftrightarrow q_0\)、\(e_1 \leftrightarrow q_L\)，路徑段數 \(L\)，使用 \(N = L+1\) 個量子位元。
- **整梯節點數** 常為 \(2(L+1)\)；競賽子問題的 \(N\) 與整梯不同，書面須區分。

### 2.2 目標態與驗證指標

- **預設製備態**：\(|\Phi^+\rangle = (|00\rangle + |11\rangle)/\sqrt{2}\)
- **理想驗證**：Statevector；CHSH \(= \sqrt{2}(\langle ZZ\rangle + \langle XX\rangle) = 2\sqrt{2}\)
- **含噪診斷**：Z 基聯合量測 \(P(00)+P(11)\)（兩端 marginal）
- **資源**：CNOT 計數、電路深度

## 3. 方法

### 3.1 策略甲：靜態 GHZ 鏈

**流程**：\(H(e_0)\) → 沿鏈正向 CNOT → 反向 CNOT 解糾纏，使僅 \(e_0\)、\(e_1\) 保留貝爾糾纏。

**CNOT 封閉式**：\(2L - 1\)

**L=2 解析推導**（三量子位 \(q_0,q_1,q_2\)）：

1. \(H(q_0)\)：\((|000\rangle + |100\rangle)/\sqrt{2}\)
2. \(CX(q_0,q_1)\)：\((|000\rangle + |110\rangle)/\sqrt{2}\)
3. \(CX(q_1,q_2)\)：GHZ \((|000\rangle + |111\rangle)/\sqrt{2}\)
4. \(CX(q_0,q_1)\) 解糾纏：\((|000\rangle + |101\rangle)/\sqrt{2} = |\Phi^+\rangle_{0,2} \otimes |0\rangle_1\)

### 3.2 策略乙：糾纏交換

**流程**：沿鏈鋪相鄰 \(|\Phi^+\rangle\) 對 → Bell 測量（\(CX\) 後 \(H\)，再 Z 基量測）→ 依古典 bit 對右側邊界做 \(X/Z\) 前饋。

**CNOT 封閉式**：\(L\)（\(L\) 為正奇數）

**實作注意**：含噪比較在 \(L=3\) 採讀取後古典解碼（與 Z 基前饋等價）；理想驗證以 Statevector 逐步塌縮完成任意奇數 \(L\)。

### 3.3 雜訊模型（公平比較）

兩策略共用 `default_fair_comparison_noise_params()`：

| 參數 | 值 | 說明 |
|------|-----|------|
| \(p_{1q}\) | 0.001 | 單量子位 depolarizing |
| \(p_{2q}\) | 0.01 | 雙量子位 depolarizing |
| \(p_{\mathrm{readout}}\) | 0.02 | 讀出翻轉 |
| \(T_1, T_2\) | 60 μs, 80 μs | 熱弛豫 |
| 古典延遲 | 2500 ns | 端點等待／前饋 |

詳見 `results/resource_formula_table.csv` 與 `figures/resources_cx_vs_L.png`。

### 4.2 含噪 Crossover（Ephys 模擬）

在公平雜訊模型下（8192 shots）：

| L | 靜態 \(P(00)+P(11)\) | 糾纏交換 |
|---|---------------------|----------|
| 1 | 0.930 | 0.901 |
| 3 | 0.919 | 0.838 |
| 5 | 0.898 | 0.784 |
| 7 | 0.880 | 0.716 |

**觀察**：短距（\(L=1\)）靜態策略略優；隨 \(L\) 增加，靜態衰減較緩、交換因 mid-circuit 量測與 idle 累積而快速下降。此與論文 Fig. 1(c)「\(n \lesssim 10\) 單元較優、長距動態優勢需更大 \(n\) 與更低 \(\lambda_{\mathrm{meas}}\)」定性一致。

圖表：`figures/crossover_p_success.png`、`figures/crossover_theory_vs_simulation.png`

### 4.3 Error Budget 統一分析（第三幕）

採用論文式 (1)：

\[
\lambda_{\mathrm{tot}} = t_{\mathrm{idle}}\,\lambda_{\mathrm{idle}} + N_{\mathrm{CNOT}}\,\lambda_{\mathrm{CNOT}} + N_{\mathrm{meas}}\,\lambda_{\mathrm{meas}}
\]

過程保真度下界：\(\mathcal{F}_{\mathrm{proc}} \geq e^{-\lambda_{\mathrm{tot}}}\)（Lemma 1）。

| 策略 | \(N_{\mathrm{CNOT}}\) | \(N_{\mathrm{meas}}\) | \(t_{\mathrm{idle}}\) |
|------|----------------------|----------------------|------------------------|
| 靜態 GHZ（\(n=L+1\)） | \(2L-1\) | 0 | \(\approx n^2/4\) |
| 糾纏交換 | \(L\) | \(L-1\) | \(1 + \mu L/2\) |
| LRCX dynamic | \(L+1\) | \(L\) | \(2 + \mu\) |
| LRCX unitary Ic | \(4L+1\) | 0 | 0 |

參數：\(\mu \approx 3.65\)，\(\lambda_{\mathrm{idle}}=0.03\)，\(\lambda_{\mathrm{CNOT}}=0.02\)，\(\lambda_{\mathrm{meas}}=0.03\)（論文 Fig. 7）。

**Crossover 條件**（論文 Appendix B.2）：動態優於單元 Ic 當 \(n\,\lambda_{\mathrm{meas}} < 3n\,\lambda_{\mathrm{CNOT}}\)，即 \(\lambda_{\mathrm{meas}} < 3\lambda_{\mathrm{CNOT}}\)。本專題模擬中 measurement 與 classical latency 仍為瓶頸，故短距未見動態全面優勢。

詳見 `results/error_budget_table.csv`、`figures/error_budget_fidelity_vs_L.png`。

## 5. 討論與限制

1. **模擬 vs 實機**：本專題以 Qiskit Aer 含噪模擬為主；論文在 ibm_sherbrooke 127Q 實機驗證。前饋延遲 \(\mu\) 在實機上可能隨 bitstring 指數增長（論文 Section III 討論）。
2. **`if_test` 穩定性**：含噪模擬對 dynamic circuit 的 `if_test` 支援不一，故含噪比較採古典解碼；理想路徑以 Statevector 驗證。
3. **問題粒度**：競賽聚焦 Bell **態製備**；論文主軸為 **閘 teleportation** 與 GHZ **態製備**——兩者物理機制相通但優化目標不同。
4. **誠實陳述**：不宜宣稱動態策略「與 \(L\) 完全無關的常數總時間」；量測、讀出、古典延遲均隨協議而變。
